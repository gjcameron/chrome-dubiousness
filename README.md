# chrome-dubiousness
Experts Are Dubious at the NYT

By Greg Cameron https://gregcameron.com

Steps for Local Installation:
* Extract the Files: If you downloaded a .zip file, right-click it and extract its contents into a dedicated folder.
* Open Extensions Page: Open Google Chrome, type chrome://extensions in the address bar, or go to the menu (three dots) > Extensions > Manage Extensions.
* Enable Developer Mode: Turn on the toggle switch for Developer mode in the top-right corner.
* Load Unpacked Extension: Click the Load unpacked button that appears on the top left.
* Select Folder: Select the folder containing the extracted extension files (the one containing manifest.json) and click Select Folder.
